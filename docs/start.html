Начало работы

Язык #Ф - функциональный языком программирования, придуманный тремя студентами Московского Авиационного Института, в качестве лабораторной работы по курсу Функциональное программирование.

Этот язык прост и понятен в использовании, а главное - вся программа на нем может быть написана на нашем любимом великом и могучем русском языке!

Установка и запуск

Для работы с языком #Ф необходимо:

  1. Cклонировать репозиторий
    git clone git@github.com:MAILabs-Edu-2024/fp-compiler-lab-average-f-enjoyer.git
  2. В VS Code установить расширение Ionide for F#
  3. Перейти в папку с проектом и запустить код файла Program.fs. Выполнится пример программы на #Ф, написанный в файле sample.x - посчитается факториал числа 5.
